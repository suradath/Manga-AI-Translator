# Manga-AI-Translator
เครื่องมือแปลมังงะครบวงจรบนเว็บ รองรับการลากคุมพื้นที่เพื่อดึงข้อความ (OCR) และแปลอัตโนมัติผ่าน OpenRouter (Gemini/ChatGPT/DeepSeek) พร้อมระบบลบข้อความเดิมและใส่พื้นหลัง (Inpainting) ผู้ใช้สามารถตรวจทานและปรับแต่ง Font, ขอบหนา รวมถึงหมุนกล่องข้อความได้อิสระ เพื่อการแปลมังงะที่รวดเร็วและมีคุณภาพ
